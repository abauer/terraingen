  • 6 - Bedrock
  • 7 - Dirt (with grass)
  • 8 - Coal
  • 9 - Dirt (with snow)
TerrainGen utlizes a block ID system. An array is created for each row. Each array consists of 40 indexes, each assigned an ID. Each ID has a related image (except 0).
